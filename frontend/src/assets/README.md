# 图片资源说明

## 需要添加的图片文件

### 1. fingerprint.jpg
- **用途**: 中央指纹图标
- **建议尺寸**: 300x300px
- **格式**: JPG
- **描述**: 科技感的指纹图案，带有蓝色发光效果，用于页面中央的指纹识别区域

### 2. tech-bg.jpg
- **用途**: 科技背景图片
- **建议尺寸**: 1920x1080px
- **格式**: JPG
- **描述**: 深色科技网格背景，带有蓝色发光线条和节点，作为整个页面的背景

### 3. logo-star.jpg
- **用途**: Logo星星图标
- **建议尺寸**: 100x100px
- **格式**: JPG
- **描述**: 发光的星星或网络节点图标，带有蓝色渐变效果，用于顶部导航栏的logo

## 如何添加图片

1. 准备符合要求的图片文件
2. 将图片重命名为对应的文件名（如 fingerprint.jpg）
3. 将图片文件放置在此文件夹中
4. 刷新前端页面即可看到效果

## 图片要求

- 文件大小建议控制在 500KB 以内
- 使用 JPG 格式以获得更好的压缩效果
- 图片应该与整体科技感设计风格保持一致
- 建议使用深色背景配合蓝色发光效果
